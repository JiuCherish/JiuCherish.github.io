截断数组（acwing）

题面：

给定一个长度为 n 的数组 a

现在，要将数组从中间截断，得到三个 非空 子数组。

要求，三个子数组内各元素之和都相等，问有几种截法。

数据范围：

$$1 \leq n \leq 10^5 , -10000 \leq a_i \leq 10000$$

题解：

如果整个数组之和 除不尽3，那么一定分割不成功，如果现在的段刚好为sum / 3那么说明这一段能分出来，用aa去记录可以分成几段，最后在找一个位置，使得sum / 3 * 2的位置即可。
